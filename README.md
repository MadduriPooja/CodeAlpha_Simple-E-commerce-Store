# 🛒 E‑Commerce Website

A professional e‑commerce web application built with **Node.js, Express, HTML, CSS, and JavaScript**.  

It includes **login & registration, product listings with images, cart management, and order placement** — all styled with a modern dark theme.

## 🚀 How to View the Site

To view the site, you should **clone this repository** to your local machine.
Once Cloned,Follow these steps

1.Open your terminal/command prompt and move into the cloned folder:
    cd ecommerce-site
    
2.Install all required Node.js packages:
    npm install
    
3.Run your backend:
    node server.js
    
4.You should see:
    Server running at http://localhost:3000
    
5.Go to:
    http://localhost:3000 
    
NOTE: Hold down Ctrl and click the link shown (http://localhost:3000) in your terminal.
